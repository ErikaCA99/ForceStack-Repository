Fundamentos de Python

¿Qué es Python?

Python es un lenguaje de programación interpretado, dinámico, multiplataforma y orientado a objetos. Fue creado por Guido van Rossum en 1991 y se caracteriza por su sintaxis limpia y legible. Su filosofía se basa en “la claridad por encima de la complejidad”, lo que lo convierte en una herramienta ideal para principiantes y expertos.

Características clave

  • Sintaxis simple y legible (fácil de mantener).
  • Lenguaje interpretado (no requiere compilación).
  • Multiplataforma (funciona en Windows, Linux y macOS).
  • Ecosistema rico en librerías: NumPy, Pandas, Django, Flask, etc.
  • Ideal para automatización, IA, ciencia de datos, web y más.

Instalación y entorno

Para comenzar a programar en Python:

  1. Descarga e instala Python desde la web oficial.
  2. Verifica la instalación:
python --version

Usa un editor como VS Code o Jupyter Notebook.

Tu primer programa

print("¡Hola, mundo!")

Este clásico “Hola Mundo” imprime texto en consola. En Python, print() es la función más básica para mostrar información.

Sintaxis, Variables y Tipos de Datos

Indentación y estructura

Python utiliza indentación (espacios o tabulaciones) para definir bloques de código. No usa llaves ni punto y coma, lo que mejora la legibilidad.

if 5 > 2:
    print("Cinco es mayor que dos")  # Correcto
else:
    print("Error de lógica")

Variables y tipado dinámico

En Python no es necesario declarar el tipo de variable. El intérprete lo infiere automáticamente.

nombre = "Alex"
edad = 22
pi = 3.1416
activo = True

Tipos de datos básicos

  • int - Números enteros
  • float - Números con decimales
  • str - Cadenas de texto
  • bool - Valores booleanos
  • list - Listas ordenadas y mutables
  • tuple - Tuplas inmutables
  • dict - Claves y valores

Conversión de tipos

edad = "23"
edad = int(edad)
print(type(edad))

Errores comunes

  • Usar mayúsculas distintas: Print() no es lo mismo que print().
  • Olvidar los dos puntos : al final de estructuras.
  • Desalinear la indentación (error frecuente en principiantes).

nombre = input("¿Cuál es tu nombre? ")
